# College Management Website - Smt T.K.R. Polytechnic

A modern, responsive college website featuring information about courses, faculty, campus life, and an interactive contact form.

## 🚀 Quick Start

1. **Install Requirements:**
   Make sure you have Python installed, then double-click the `start_server.bat` file.
   Alternatively, run: `pip install -r requirements.txt`

2. **Run the Server:**
   The `start_server.bat` file will also start the server. 
   Alternatively, run: `python app.py`

3. **View the Website:**
   Open your web browser and go to: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 📁 Project Structure

- `app.py`: Flask backend server that serves the website and handles form submissions.
- `index.html`: The main website page.
- `styles.css`: Styling for the website.
- `script.js`: Interactive features and form submission logic.
- `Database.txt`: Stores all contact form submissions.
- `start_server.bat`: A handy one-click startup script for Windows.
